Magnolia is an open-source enterprise class Content Management System
developed by Magnolia International Ltd., based on the standard API
for Java Content Repositories (JCR).

This package contains:

   Magnolia Core 6.2.17
   Magnolia empty webapp 6.2.17
   Magnolia empty webapp 6.2.17
   Magnolia Internationalization Framework 6.2.17
   Magnolia JAAS 6.2.17
   magnolia-ldap 1.10.3
   magnolia-license 1.7.4
   Magnolia Link Unfurl Module 1.3.10
   Magnolia Backup Module Project 2.4.2
   Magnolia Mail module 5.5.10
   magnolia-module-rssaggregator 2.6.4
   magnolia-module-soft-locking 3.0.1
   Magnolia Rendering 6.2.17
   Magnolia Templating 6.2.17
